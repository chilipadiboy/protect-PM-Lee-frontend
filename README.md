## Developer Guide

### Frontend
To run the front-end, go to 'healthcare-app-client' and run:
1. `yarn install`
1. `yarn start`

### Backend
To run the server, go to the backend directory and run `gradlew.bat bootRun` (or `./gradlew bootRun` for Mac/Linux).
